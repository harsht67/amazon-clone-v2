# Amazon Clone 

## Features 
- Responsive Design 
- Login/Signup 
- Cart 

## Built with 
- React 
- Sass => for styling
- Redux 
- Sanity => for back-end 
- Firebase => for user authentication 
- React icons 
- Netlify => to deploy project 

## Upcoming features

- Wishlist => can create/edit/delete wishlist 
- Previous orders => users can see their previous order
